Referencia de API para la directiva oCustomContent de Obelisco

OCustomContentDirective

Directiva estructural: *oCustomContent

La directiva OCustomContentDirective permite definir contenido personalizado de manera dinámica dentro de los componentes:

  • Banner
  • Bloque de trámite
  • Colapsable
  • Destacado
  • Footer
  • Header
  • Mapa
  • Mensaje de estado
  • Modal
  • Panel
  • Tarjetas
